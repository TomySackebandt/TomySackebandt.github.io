Projet 2

Projet de moteur de jeu de type "visual novel" sur une "fantasy console" :

Lien vers la page principale du projet RomanTIC est un projet personnel où j'ai crée un moteur de jeu pour les "visual novels". Il y a plusieurs exemples qui existent déjà pour ce type de moteur, le plus connu étant renpy.
La différence principale étant l'utilisation de l'environnement de "fantasy console" TIC80 qui utilise le langage de programmation Lua. Le projet est dans sa phase finale, je suis en train de fournir une documentation sur la manière d'utiliser les outils nécessaires à la création d'un "visual novel"

Liste des compétences sollicitées :
  • Création d'une documentation
  • Développement d'application : Maitrise du langage Lua
