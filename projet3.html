Projet 3

Projet de création d'un site pour une entreprise fictive :

Au cours de ma formation, nous avons eu comme projet de créer en binôme une base de données ainsi que des applications pour y accéder (web et en c#).
La base de données devait fournir les informations nécessaires à la gestion d'employés et de la gestion de note de frais. Les modifications importantes devaient être accessible depuis l'application créée en C# et les créations de note de frais par le web. J'étais responsable du développement de la partie web.

Liste des compétences sollicitées :
  • Création d'une documentation
  • Création d'une base de données
  • Suivi d'un cahier des charges
  • Développement d'application web : html/css, js, php
